# Data-Driven-Demand-Planning-EOQ

## Project Objective

The goal of this project is to improve the classical EOQ inventory model by integrating demand forecasting. Historical retail sales data will be used to estimate demand, while holiday data will be included as an external factor. The forecasted demand will then be used to calculate an adaptive EOQ model.
