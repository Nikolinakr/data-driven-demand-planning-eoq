# Data-Driven-Demand-Planning-EOQ

## Project Objective

The goal of this project is to improve the classical EOQ inventory model by integrating demand forecasting. Historical retail sales data will be used to estimate demand, while holiday data will be included as an external factor. The forecasted demand will then be used to calculate an adaptive EOQ model.

## Data Sources

1. Online Retail Dataset  
Due to file size limitations, the dataset is not included in the repository.  
It can be downloaded from:  
https://www.kaggle.com/datasets/carrie1/ecommerce-data?resource=download

2. Holidays Dataset  
Included in the repository.

## How to run

1. Clone the repository.
2. Install dependencies:
   pip install -r requirements.txt
3. Download the Online Retail dataset from Kaggle.
4. Save it as data/raw/online_retail.csv.
5. Run the notebook from the notebooks folder.
